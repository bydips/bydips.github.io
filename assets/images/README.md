# Assets Images

Simpan gambar lokal di folder ini (contoh: hero-home.jpg, katalog-barber.webp, founder.webp).

Panduan singkat:
- Gunakan format `.webp` atau `.jpg` untuk performa.
- Target ukuran: 1600px sisi panjang untuk hero, 960px untuk kartu katalog.
- Setelah menambah file, update path `src` di halaman terkait.
